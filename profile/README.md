## Sobre nós

Somos a equipe da **Escola de Magia e Bruxaria de Hogwarts**, que é dedicada a criar uma ferramenta digital para controle de turmas da escola. Nossa missão é digitalizar o gerenciamento e geração de boletins para os professores, bem como oferecer dashboards inteligentes sem nenhuma perda ou omissão de dados.

## Principais funcionalidades

O sistema foi projetado para ser completo e intuitivo. Veja o que ele oferece:

*  **Buscar aluno pela matrícula**: Monitorar nome do aluno, matrícula, notas lançadas e observações.
*  **Lançar notas das disciplinas**: Que são contempladas no sistema de boletim.
*  **Enviar observações para o aluno**: Que podem ser visualizadas em conjunto com outros dados.
*  **Visualizar boletim escolar**: Em conjunto com a situação do aluno.
*  **Recuperação de senha**: Prevenindo erros do usuário.
